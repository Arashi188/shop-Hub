# LUXE - Premium E-Commerce Website

## Overview
A modern, responsive e-commerce product showcase website with WhatsApp integration, shopping cart, and hero slideshow.

## Features
- ✅ Hero slideshow with auto-play and manual controls
- ✅ Product grid with category filtering
- ✅ WhatsApp pre-filled messages per product
- ✅ Shopping cart with localStorage persistence
- ✅ Product detail pages
- ✅ Responsive design (mobile-friendly)
- ✅ Sticky header and smooth scroll animations
- ✅ Touch swipe support for hero slider
